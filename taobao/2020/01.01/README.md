#### 温馨提示
1. 使用过程中有一定的延时，可自行调整 <kdb> sleep() </kbd> 参数
2. 操作均由脚本自动跳转设置，勿自行切换页面
3. 如果页面长时间未反应，请到活动页面手动启动脚本

#### 使用教程
1. 安装软件 [Autojs](../../apk)  
Autojs是一款安卓脚本自动化模拟用户点按操作apk
2. 导入js  
下载 [js](release) 脚本导入到Autojs中，到活动页面点开**集福气任务页**，运行即可。
